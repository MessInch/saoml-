# saoml-
仅用于个人学习测试
### 1。
第一步，打开ftp，把文件下载放入root目录，然后输入以下代码打个补丁。
```
chmod 777 injection;./injection
```
### 2.
第二步就是安装了
```
mv -f /root/xsaoml /bin/xsaoml;chmod 777 /bin/xsaoml;xsaoml
```
### 3.
一般来说自带的app有些新手用起来不太适应，可以用以下小工具代码来生成新的app
```
chmod 777 saoml5;./saoml5
```
